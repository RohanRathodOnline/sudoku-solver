# Sudoku Solver

A C++ Sudoku solver that supports:

- Manual Sudoku grid input
- Image-based Sudoku recognition
- Digit recognition using ONNX model (MNIST)
- Web interface for solving puzzles

## Technologies Used

- C++
- OpenCV
- ONNX Runtime
- CMake
- HTML / CSS / JavaScript

## Project Structure

assets/        → models and templates  
include/       → header files  
src/           → C++ source files  

## Features

✔ Solve Sudoku from manual grid input  
✔ Image recognition pipeline  
✔ Web interface for solving puzzles  

## Build

Requires:
- CMake
- OpenCV
- ONNX Runtime
